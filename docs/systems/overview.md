# Furut System Overview

Furut is a platform for managing outdoor stays and experiences.

## Core Building Blocks

- Account → Who owns things
- Property → Physical location
- Unit → Bookable space inside property
- Experience → Activities hosted at property

## System Flow

Account
  → Property
      → Units
      → Experiences

## Philosophy

- Everything is modular
- Units define behavior (stay, event, etc.)
- Rules are configurable, not hardcoded
